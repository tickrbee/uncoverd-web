# App Screenshots

Place your app screenshots in this directory with the following filenames:

1. **discover-swipe.png** - Screenshot showing the stock discovery/swipe interface (like the Sanderson Design Group screen)
2. **ai-insights.png** - Screenshot showing AI-powered insights and analysis (can use the same discover screen or a detail view)
3. **rankings.png** - Screenshot showing the global rankings/leaderboard screen
4. **performance.png** - Screenshot showing performance tracking and investor profile (like the "Performance vs All Users" screen)
5. **top-picks.png** - Screenshot showing the Top Picks screen with crowd discoveries

Recommended dimensions: 400x800px (mobile portrait) or similar aspect ratio.
Formats: PNG, JPG, or WebP
